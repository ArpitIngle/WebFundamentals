# Document doesn't have a valid hreflang  {: .page-title }

## Overview {: #overview }

## Recommendations {: #recommendations }

Define an `hreflang` for each language version of a URL. For example, suppose that you have 3
versions of a page:

* An English version hosted at `https://example.com`.
* A Spanish version at `https://es.example.com`.
* A German version at `https://de.example.com`.
